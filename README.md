# Superfluid Dapp

1. Switch to Goerli test network in meta mask
2. Deposit token in superfluid dashboard to get super wrapped token (e.g. xETH)
    - https://app.superfluid.finance/
3. Run yarn start
4. Connect wallet to start and stop streams.
